# Daily Lunch Menu Website

This website is designed to present the daily menu of several restaurants using Python. It provides users with an easy way to check the menu options available at different restaurants for their lunch.

## Features

- Display the daily menu of multiple restaurants
- Filter menu options based on dietary preferences or restrictions
- Search for specific dishes or ingredients
- User-friendly interface for easy navigation

## Technologies Used

- Python
- HTML/CSS
- Flask framework
- SQLite database

## Installation

1. Clone the repository: `git clone https://github.com/your-username/daily-lunch-menu.git`
2. Navigate to the project directory: `cd daily-lunch-menu`
3. Install the required dependencies: `pip install -r requirements.txt`
4. Run the application: `python app.py`
5. Open your web browser and visit `http://localhost:5000` to access the website.

## Usage

1. Upon accessing the website, you will see a list of restaurants and their daily menu options.
2. Use the search bar to find specific dishes or ingredients.
3. Use the filters to narrow down the menu options based on dietary preferences or restrictions.
4. Click on a restaurant to view more details about their menu.
5. Enjoy exploring the daily lunch options and plan your meals accordingly!
